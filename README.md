Sistema completo de anamnese digital em formato wizard, desenvolvido para personal trainers coletarem informações dos alunos de forma intuitiva e eficiente.

## ✨ Funcionalidades

### 📱 Interface do Usuário
- **Wizard de 10 passos** - Navegação intuitiva passo a passo
- **Temas Claro/Escuro** - Toggle com persistência da preferência
- **Acessibilidade 50+** - 3 tamanhos de fonte (A-, A, A+)
- **Design Responsivo** - Funciona perfeitamente em mobile
- **Animações Suaves** - Transições e feedbacks visuais

### 💾 Persistência de Dados
- **Auto-save** - A cada 30 segundos e ao mudar de passo
- **MySQL** - Armazenamento seguro no banco de dados
- **Histórico** - Todas as avaliações são mantidas
- **Exportação JSON** - Dados prontos para integração

### 🔌 Integrações
- **Webhook Configurável** - Envia JSON via POST para URL externa
- **Endpoint de Teste** - Valide a integração antes de usar
- **Formato Padrão** - Dados estruturados para fácil consumo

### 🎨 Design System
- **Cores Personalizáveis** - Destaques em laranja
- **Ícones Ilustrativos** - Melhor experiência visual
- **Cards e Modais** - Interface moderna e limpa
- **Feedback Visual** - Notificações e indicadores

### Pré-requisitos
- PHP 8.1 ou superior
- Composer
- MySQL 5.7 ou superior
- Node.js e NPM (opcional)
